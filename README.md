# HYSquareIndicator

####方块活动指示器



####开启

- (IBAction)start:(UIButton *)sender {
    
    
    [HYSquareIndicatorView show];
    
    
}

####关闭

- (IBAction)dismiss:(UIButton *)sender {
    
    
    [HYSquareIndicatorView dismiss];
}
